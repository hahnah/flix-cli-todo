# Flix CLI Todo

CLI Todo Application made by Flix language.

## Setup

### Install JDK

Install JDK from [https://adoptium.net/ja/temurin/releases](https://adoptium.net/ja/temurin/releases).

### Install Flix

At this project's root directory,  
run `curl -L https://github.com/flix/flix/releases/latest/download/flix.jar -o flix.jar`.

## Run application

```
java -jar flix.jar run
```
